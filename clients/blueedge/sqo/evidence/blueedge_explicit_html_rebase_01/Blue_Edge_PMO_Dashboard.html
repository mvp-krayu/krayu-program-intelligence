Data Governance, Privacy & Compliance
🔐 Data Classification & Sensitivity
4-tier data classification applied to all 161+ metrics
RESTRICTED
Driver biometric data (face, PERCLOS), health data, personal location history, ADR gas exposure records. GDPR Article 9 special category. Encrypted at rest + in transit. Access: compliance officer only.
CONFIDENTIAL
Vehicle GPS history, driver scoring, cargo manifests, fuel consumption, financial data. GDPR personal data. Role-based access. AES-256 at rest. Audit logged on every access.
INTERNAL
Fleet KPIs, aggregated analytics, maintenance records, alert histories. Non-personal. Tenant-scoped access. Standard TLS in transit.
PUBLIC
GTFS static feeds, vehicle DIDs (public keys only), anonymised ridership statistics, compliance certificates. Publicly accessible via API.

🔬 Anomaly Detection Pipeline
Statistical Z-Score (Real-time)
Rolling 5-minute window per metric per vehicle. Flag values >3σ from vehicle baseline. Auto-calibrates baseline on data quality events.
Isolation Forest (Kinesis Analytics)
Multivariate anomaly detection across correlated sensors (e.g. fuel level + engine load + GPS speed). Flags complex anomalies missed by single-metric thresholds.
Cross-Sensor Consistency Check
GPS-derived speed vs J1939 wheel speed vs accelerometer integration. Tank level delta vs flow meter reading. Inconsistency >5% triggers data quality alert.
Sensor Fault Classification (NPU)
Edge ML model on SVG NPU classifies each sensor as OK / Degraded / Stuck / Failed based on signal characteristics. Bad sensor data quarantined before cloud upload.
