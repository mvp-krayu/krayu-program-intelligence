# Execution Signal Model

## Definition

An execution signal is an interpretable condition derived from observable execution evidence.

Metrics alone describe measurements. Signals describe meaning.

Example:

Metric: Sprint velocity  
Signal: Throughput instability derived from velocity variance

## Signal Structure

Signal Family → Signal Class → Signal Instance

Example:

Delivery Stability → Throughput Instability → Velocity variance exceeding threshold.

## Signal Lifecycle

1. Observation of execution event
2. Evidence capture
3. Signal detection
4. Signal scoring
5. Trend evaluation
6. Pattern aggregation

Signals are the atomic building blocks of Program Intelligence.
