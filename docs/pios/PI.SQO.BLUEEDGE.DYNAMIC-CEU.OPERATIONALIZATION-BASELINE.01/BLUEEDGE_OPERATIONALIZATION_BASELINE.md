## 5. Operationalization Safety Envelope

### 5.1 Safe Activation Principle

Overlay activation may evolve QUALIFICATION STATE.
Overlay activation MUST NEVER evolve SUBSTRATE TRUTH.

```
BEFORE overlay:  S2, Q-02, backed=4/17, readiness=0.133
AFTER overlay:   S2→S3 possible, Q-02→Q-01 possible, backed=4+N/17

Substrate:       UNCHANGED (topology, DPSIG, decision validation, reproducibility)
```

## 7. Baseline Immutability Contract

The BlueEdge certified baseline (§3) is the IMMUTABLE reference state
for all operationalization:

1. **No overlay may modify any baseline artifact.**
2. **Every overlay is measured against this baseline.**
3. **Revoking all overlays restores this exact baseline evaluation.**
4. **Historical replay uses this baseline as Layer 0.**
5. **If the substrate is re-executed (new pipeline run), the baseline
   changes — but that is a pipeline event, not an overlay event.**

This contract ensures that operationalization is always reversible:
the certified baseline is the deterministic anchor point.
