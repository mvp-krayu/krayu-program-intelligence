## 5. STRUCTURAL NOTE — SIG-003 ELSE BRANCH

The else branch of `compute_sig_003` is labeled "Unreachable in static context" in source.
It returns BLOCKED with no formula. This is not a comment artifact — it is an executed
code path with no computation. If VAR_AT_001 and VAR_AT_002 were provided as non-UNDEFINED
values, the function would enter the else branch and return BLOCKED with no output anyway.

This is a distinct failure from data absence. It is documented separately in
`signal_failure_classification.md`.
