## Center of Gravity

**GOVERNED COGNITION** — the system's ability to perform deterministic cognitive acts on structural evidence under bounded authority.

Governed Cognition = 22 cognitive functions × 5 cognition strata.

Everything else serves it:
- Spine exists so governed cognition has **memory**
- Chronicle exists so governed cognition can **prove itself**
- PICP exists so governed cognition can be **packaged**
- SQO exists so governed cognition can **earn authority**
- Personas exist so governed cognition can **reach audiences**

PI cognition operates across three orthogonal dimensions:

| Dimension | What it represents | Key structures |
|---|---|---|

| **Temporal Continuity** | What the system REMEMBERS | Spine (8 object classes), Chronicle (append-only event lifecycle), Certification (62-check replay proof) |
| **Structured Packaging** | What the system DELIVERS | CIP (L0–L3), PICR (L4 materializers), PICP (L4 package — 9 objects), PRE (L5 projection) |

## PICP Positioning

PICP is a **cognition container** — a transport mechanism that packages the current state of structured cognition for downstream consumption. It is an artifact PRODUCED BY the architecture, not the architecture itself.

**Disappearance test:** If PICP vanished, all 22 cognitive functions, Spine, Chronicle, SQO, signal derivation, condition synthesis, consequence compilation, topology cognition, persona projection, and LENS runtime continue operating. What is lost: portable, diffable, versionable cognition snapshots.

**Positioning:** One of four satellite systems serving Governed Cognition. The Packaged Cognition satellite.
