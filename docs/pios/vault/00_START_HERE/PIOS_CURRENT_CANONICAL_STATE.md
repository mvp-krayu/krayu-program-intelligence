## Three-Tier Authority Model

```
DETERMINISTIC → INVESTIGATIVE → INTERPRETIVE
     ↑               ↑               ↑
  rendering    guided traversal   bounded synthesis
  no selection  topology-derived   evidence-bound
  pure display  structural adjacency  75.x required
```

- **DETERMINISTIC:** Pure rendering. No selection. Display exactly what the payload contains.
- **INVESTIGATIVE:** Topology-derived guided traversal. The system selects which structural connections to surface based on navigation position. Selection criteria are topology-derived, not inference-derived. L6/L7 consumption. No 75.x required.
- **INTERPRETIVE:** Bounded synthesis under explicit 75.x authorization. Evidence-bound. Disclosure-wrapped.

**Investigative authority** is the new middle layer introduced by Phase 5 roadmap. It permits surfacing connected domains via topology edges, exposing pressure zones via structural adjacency, traversing evidence lineage, presenting temporal deltas. It does NOT permit inference, recommendation, prioritization, or prediction.
