## Quick Orientation

The Krayu Program Intelligence system is a **governed semantic qualification engine** that:

1. **Ingests** evidence from operator-provided sources
2. **Reconstructs** semantic domain intelligence via deterministic extraction
3. **Hydrates** the reconstruction into an operationally useful model (HYDRATED state)
4. **Qualifies** the semantic maturity via the SQO state machine
5. **Renders** executive-grade intelligence with honest qualification disclosure
