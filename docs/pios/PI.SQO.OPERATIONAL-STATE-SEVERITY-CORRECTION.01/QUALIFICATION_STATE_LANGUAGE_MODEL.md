# Qualification State Language Model

## Normalized Terminology

| Old | New | When |
|-----|-----|------|
| BLOCKED | QUALIFICATION INCOMPLETE | projection blockers present |
| BLOCKED | EXPANSION CONSTRAINED | expansion blockers only |
| Critical Blockers | Qualification Blockers | Class A projection debt |
| Critical Blockers | Expansion Gaps | Class C expansion debt |
| X blocking | X progression constraints | all contexts |
| X% readiness | readiness label | progression display |

## Progression Readiness Labels

| Range | Expansion Context | Qualification Context |
|-------|------------------|----------------------|
| 0% | grounding readiness not yet established | qualification prerequisites incomplete |
| 1-24% | grounding readiness emerging | qualification prerequisites incomplete |
| 25-74% | grounding readiness progressing | qualification readiness progressing |
| 75-100% | grounding readiness approaching | qualification readiness approaching |

## Forbidden Language

- "Critical" for expansion debt
- "Blocked" for expansion-constrained states
- "Failure" for progression limitations
- Any AI/recommendation language
