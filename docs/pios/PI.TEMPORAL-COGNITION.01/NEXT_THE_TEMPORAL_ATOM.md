## The Unexamined Assumption

PCD-010 and the proof-gate work assumed the unit that evolves over time is the **whole Answer Object**: AO-011 → TV-001, AO-003 → TV-003.

This was never examined. It is an assumption, not a finding.

**Open question: what is the temporal atom — what does the runtime actually observe through time?**

Candidate atoms (not yet discriminated):

- **Answer Object State** — the object's verdict value (divergence: yes/no; the whole object as a unit).
- **Answer Object Confidence** — not whether the finding holds, but how strongly. Confidence can move while the verdict is static.
- **Answer Object Qualification** — the evidence basis / governance level. The finding may be unchanged while its qualification matures (S1→S2). This is itself a temporal trajectory — of trust, not of structure.
- **Answer Object Topology** — the domains/edges the object spans. The divergence could hold while *which domains* diverge shifts underneath it.

These are not the same atom, and they have different proof requirements:

- A change in **State** requires changed specimen (the proof gate we just locked).
- A change in **Qualification** can occur with a *stable* specimen as evidence accumulates — which would land in a different cell of the proof-gate matrix and may be observable without a second commit.
- A change in **Confidence** or **Topology** may decompose AO-011 into sub-fields that each have their own trajectory.

If the temporal atom is qualification or confidence rather than state, then some temporal cognition may be provable *without* a second specimen commit at all — because qualification evolves under a stable specimen as evidence is added.

## Why This Does Not Need a Second BlueEdge Run

Determining the temporal atom is a decomposition question, not a measurement question. It asks: *when an Answer Object "changes," what exactly is the thing that changed?* That can be reasoned from the structure of existing Answer Objects (AO-001..AO-020, AO-011) — what fields they carry, which of those fields are state vs confidence vs qualification vs topology — without observing any of them twice.

The proof gate bought the freedom to answer this properly. Hunting for a second BlueEdge run now would be premature: we would be measuring the trajectory of an atom we have not yet identified.
