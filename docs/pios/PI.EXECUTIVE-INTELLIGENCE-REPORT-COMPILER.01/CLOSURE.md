**New Concepts Introduced:**
1. **Executive Intelligence Compiler (EIC)** — L4 transformation system converting governed structural evidence + synthesized conditions + compiled consequences + cognition ontology into multi-format executive intelligence deliverables. Positioned at L4 in the PI pipeline, consuming L0–L3.5 outputs.

2. **Compiled Intelligence Package (CIP)** — the formalized input contract for the EIC: fullReport + synthesisResult + consequenceResult + boardroomProjection + balancedProjection + cognitionOntology + classRiskLabels + qualificationPackage.

3. **Transformation Taxonomy (T1–T7)** — seven-type classification of transformations used in executive intelligence production:
   - T1: Evidence Extraction (deterministic)
   - T2: Arithmetic Derivation (deterministic)
   - T3: Pipeline Synthesis (deterministic)
   - T4: Cross-Object Correlation (deterministic, rules needed)
   - T5: Pattern Characterization (partially deterministic)
   - T6: Audience-Specific Narration (partially deterministic)
   - T7: Consulting Judgment (not deterministic)

4. **55/20/25 Composition Ratio** — the discovered composition of executive intelligence: 55% structurally derived (fully reproducible), 20% vocabulary-driven narration (templateable), 25% consulting craft (human intelligence).

5. **Seven Cognitive Functions (CF-1 through CF-7)** — the implicit cognitive functions performed during executive intelligence production: Evidence Harvester, Intelligence Synthesizer, Narrative Architect, Finding Narrator, Differentiator Analyst, Recommendation Synthesizer, Synthesis Compressor.
