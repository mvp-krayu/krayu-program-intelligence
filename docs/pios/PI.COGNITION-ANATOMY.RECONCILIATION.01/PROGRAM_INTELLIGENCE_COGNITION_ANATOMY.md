Program Intelligence cognition operates across three orthogonal dimensions:

**DIMENSION 1: COGNITIVE FUNCTION** — What the system does

22 cognitive functions organized in 5 cognition strata. These are the acts of cognition — emergence orchestration, pressure concentration detection, evidence boundary qualification, temporal cognition, guided cognition, etc. The 22 functions are the system's cognitive repertoire.

**DIMENSION 2: TEMPORAL CONTINUITY** — What the system remembers

Spine (8 object classes) + Chronicle (governed event lifecycle) + Certification (replay proof). These carry cognition across time. A single pipeline run produces artifacts; continuity objects record HOW those artifacts evolved across runs.

PICP (9 cognition objects produced by PICR from CIP). A point-in-time, deterministic, projection-independent snapshot of structured cognition. Consumed by PRE for audience-specific rendering.

### 3.3 The Relationships

**Cognitive Functions → PICP:** The 22 cognitive functions are the system's cognitive acts. PICP is the packaging of SOME of their outputs. Not all 22 functions produce PICP objects — many produce ephemeral projection, orchestration state, or continuity mutation. The 9 PICP cognition objects are the subset of cognitive output that is deterministic, evidence-bound, audience-independent, and structurally novel.

**Spine → PICP:** Spine is the temporal memory of the system. PICP is a point-in-time snapshot. Spine feeds PICP through the CIP: `spine_objects` → `GoverningNarrativeComposer` → `governed_narrative` → `fullReport` (CIP) → PICR → PICP. Spine is UPSTREAM of PICP. A series of PICPs over time would be captured BY the Spine (as `executive_projection_snapshots`).

**Chronicle → PICP:** Chronicle provides governance proof that feeds into `structural_posture.qualification.certification`. Chronicle is evidence that the PICP was earned, not merely computed. Chronicle is GOVERNANCE for PICP.

**Personas → PICP:** Personas are not consumers of PICP. Personas are cognitive projection architectures that each instantiate the 22 cognitive functions at different depths. PICP is one input to persona projection — not the sole input. Personas also consume synthesis results, consequence compilation, topology, structural enrichment, and orchestration state directly.

## 5. WHAT PICP ACTUALLY IS

PICP is a **cognition container** — a transport mechanism that packages the current state of structured cognition for downstream consumption.

PICP is **NOT** the architecture. It is an **artifact produced by** the architecture.

### Evidence for this claim:

**Test 1: What survives if PICP disappears?**

If PICP vanished tomorrow:

| What survives | Why |
|---------------|-----|
| 22 cognitive functions | They execute regardless of packaging |
| Spine (8 object classes) | Spine carries temporal continuity — it doesn't need PICP to exist |
| Chronicle | Governed event lifecycle is independent of cognition packaging |
| SQO | Qualification lifecycle operates on governance artifacts, not PICP |
| Signal derivation | DPSIG, PSIG, ISIG compute from structural evidence |
| Condition synthesis | SignalSynthesisEngine produces conditions without PICP |
| Consequence compilation | ConsequenceCompiler maps conditions to consequences without PICP |
| Topology cognition | 9-slice overlay system operates on conditions, not PICP objects |
| Persona projection | Personas consume fullReport, synthesis results, consequences — not PICP |
| LENS runtime | All zones render from resolved payload, which is CIP, not PICP |

What is lost: the ability to produce a PORTABLE, DIFFABLE, VERSIONABLE cognition snapshot that exists independently of any consumer. PICP adds archival, temporal comparison, and external delivery capability.

**Test 2: What dies if the 22 cognitive functions disappear?**

Everything. No emergence orchestration → BALANCED is dead. No pressure concentration detection → BOARDROOM is blind. No temporal cognition → SQO is snapshots without trajectory. No guided cognition → DENSE has no interrogation paths. The 22 functions ARE the cognition.

**Test 3: What dies if the Spine disappears?**

The system loses temporal memory. Each pipeline run produces results, but there is no continuity between runs. No evolution tracking. No hero moments. No governed narrative anchored in lifecycle events. The system becomes amnesic — it computes correctly but forgets everything between runs.

### Conclusion: PICP is Layer 4 Packaging

PICP is the system's ability to say: "Here is what I currently know, in a structured, deterministic, portable format."

The architecture is the system's ability to KNOW — to perform governed cognitive acts on structural evidence, remember what it learned, prove how it earned its state, and project that cognition through audience-appropriate surfaces.

**In the anatomy:** The EIR is a historical artifact. It proved that governed cognition could produce executive intelligence. Its architectural contribution was decomposed into CIP, PICR, PICP, and PRE. The Chronicle is a living governance mechanism. They occupy different architectural roles and the relationship is historical, not structural.
