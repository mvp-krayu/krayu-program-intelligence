## Backward Compatibility Rule

Every rebinding has a two-branch structure:
1. **Governed lifecycle available** (`isGovernedS1Plus === true`) → Governed intelligence projection
2. **No governance lifecycle** → Current behavior unchanged

Detection:
```javascript
const gl = fullReport && fullReport.governance_lifecycle;
const hasGovernedLifecycle = gl && gl.available && gl.s_level;

const isGovernedS1Plus = hasGovernedLifecycle
  && ['S1', 'S2', 'S3'].includes(gl.s_level);
```

Run compatibility:
- BlueEdge genesis_e2e_03 (`promotion_state` present, `s_level: 'S2'`) → governed path
- BlueEdge productized_01_fixed (no `promotion_state`) → legacy path
- NetBox (no `promotion_state` in manifest) → legacy path
- StackStorm (no `promotion_state` in manifest) → legacy path
- FastAPI, Flask (no `promotion_state`) → legacy path
