## Offer 1: Signäl Structural Assessment (SA)

**The primary commercial wedge.**

### The Pitch

"We reveal what your code structure means for execution — fragility, coupling inertia, dependency amplification. Structural consequences that no existing tool, dashboard, or consultant can show you. In days, not months."

### Who It's For

| Buyer | Why They Buy |
|-------|-------------|
| **CTO / VP Engineering** | "I know we have structural bottlenecks. I can feel it. I need structural proof — not opinions, not code quality scores. What does the structure actually tell us about execution risk?" |
| **Transformation Director** | "We're planning a modernization. Before we start moving things, I need to see where coupling inertia resists evolution and where execution fragility concentrates." |
| **PMO Director** | "My RAG reports are opinions. I need structural evidence I can take to the steering committee. Something governed and defensible." |

### What They Get

| Deliverable | Description |
|-------------|-------------|
| **LENS Access** | Interactive structural intelligence surface. Two cognitive modes: BOARDROOM (executive consequence projection) and BALANCED (operational dynamics briefing). Topology exploration, structural condition findings, Software Intelligence consequence analysis. |
| **Structural Verdict** | 9-chapter governed document: Executive Brief, Program Overview, Structural Topology, Program Intelligence Findings, Software Intelligence Assessment, Execution Risk Landscape, Operational Ceiling, What Traditional Analysis Cannot Discover, Executive Verdict. Each finding named, severity-classified, evidence-traced. Board-ready. |
| **Evidence Record** | Governed structural evidence snapshot: topology capture, posture assessment (PROCEED / INVESTIGATE / ESCALATE), confidence envelope, governance boundary disclosure. Auditable. Archivable. |
| **Advisory Delivery Session** | Walkthrough with Signäl team. Topology → findings → consequences → verdict. 60-90 minutes. |

### What the Software Intelligence Module Reveals

9 behavioral consequence surfaces, each with severity, operational summary, consequence chain, and affected domains:

| Surface | What It Detects |
|---------|----------------|
| Execution Fragility | Regions structurally vulnerable to change — single modifications cascade unpredictably |
| Dependency Amplification | Components where the true cost of changes is systematically underestimated |
| Coupling Inertia | Coupling patterns that actively resist architectural evolution |
| Propagation Asymmetry | Unbalanced dependency flow creating cascade risk |
| Structural Gravity | Oversized components attracting disproportionate operational stress |
| Execution Constriction | Bottlenecks constraining execution throughput |
| Boundary Divergence | Boundaries drifting from intended architecture |
| Coordination Fragility | Regions where multi-party coordination is structurally brittle |
| Compound Convergence | Multiple independent structural conditions reinforcing each other |

### Engagement Model

| Phase | Duration | What Happens |
|-------|----------|-------------|
| Evidence intake | Day 1 | Customer provides repository access, dependency exports, architectural artifacts |
| Structural analysis | Day 2-3 | Signäl ingests evidence, runs structural intelligence pipeline |
| Intelligence delivery | Day 3-4 | LENS provisioned, Structural Verdict generated, Evidence Record generated |
| Advisory session | Day 4-5 | Walkthrough with customer. Findings delivered. Artifacts handed over. |

### Pricing

Per-program, per-engagement fee. Advisory-grade. Not SaaS. Not subscription. A structural intelligence deliverable.
