## Frequently Asked Questions

**"How is this different from SonarQube?"**
SonarQube measures code quality — bugs, vulnerabilities, code smells. Signäl reveals structural execution consequences — fragility, coupling inertia, dependency amplification, operational ceiling. Different input, different output, different category. Clean code in a structurally fragile region is still operationally risky.

**"We already have an architecture review process."**
Architecture reviews produce expert opinions, informed by experience. Signäl produces governed structural intelligence, derived from evidence. The Structural Verdict complements your architect's judgment with structural proof. Most architects tell us: "I knew this was a problem — now I can prove it."

**"Is this an AI tool?"**
The intelligence layer is entirely deterministic — same evidence produces the same structural findings every time. AI participates only in calibrated audience projection (~14% of the output), and every AI contribution is disclosed and governance-bounded. This is structural derivation, not AI generation.

**"What do you need from us?"**
Repository access (read-only), dependency export files, and any architectural documentation you have. We handle the rest. Evidence intake typically takes one session.

**"What if our findings are negative?"**
Every program has structural conditions — that's not a judgment, it's structural reality. Signäl classifies severity and shows where conditions concentrate. Some conditions are cosmetic. Some are systemic. The Structural Verdict distinguishes between them so you invest where it matters.

**"Can we run this on multiple programs?"**
The Structural Assessment covers one program per engagement. For ongoing access across multiple programs, see Signäl Continuous Intelligence (annual subscription) or Signäl Enterprise Intelligence (platform license).

**"What happens after the assessment?"**
Many customers convert to Continuous Intelligence — ongoing structural access with multiple assessment runs, full-depth investigation, and qualification progression tracking. The assessment shows your structural reality; Continuous Intelligence lets you track how it evolves after investment.
