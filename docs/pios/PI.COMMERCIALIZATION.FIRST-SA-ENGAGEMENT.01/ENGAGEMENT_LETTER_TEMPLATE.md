---

### 7. Deliverable Governance

The Structural Assessment Package is produced by Signäl's governed intelligence pipeline.

- **Deterministic:** Same evidence produces the same findings. The assessment is reproducible.
- **Evidence-traced:** Every finding traces to structural evidence in the provided codebase.
- **Governed:** All outputs operate within defined governance boundaries. Confidence levels and governance limits are disclosed within the deliverables.

The Structural Verdict contains named structural findings with severity classifications. Findings reflect structural conditions identified in the evidence. They do not constitute recommendations for specific remediation actions, personnel assessments, or organizational judgments.

---

### 8. Limitation of Scope

This engagement assesses the structural topology and execution consequences of the defined program. It does not include:

- Security vulnerability assessment or penetration testing
- Code quality scoring or linting
- Performance profiling or load testing
- Organizational assessment or team evaluation
- Remediation implementation or code changes
- Ongoing monitoring (available via Signäl Continuous Intelligence)
