PROGRAM INTELLIGENCE HANDBOOK

Purpose

This handbook is the synthesis and navigation layer of the Program Intelligence discipline.

It connects the intellectual outputs produced across the Program Intelligence project streams and presents them as a coherent, navigable publication.

The handbook does not replace authoritative stream artifacts.
Instead, it provides the reader with an integrated path through the discipline and links to the detailed materials stored in the repository.


HANDBOOK ENTRY MODEL

This handbook should be read as the root navigation document of the discipline.

Each section below links to a handbook artifact or to the authoritative stream domains and artifact indexes that underpin the corresponding part of the discipline.

The handbook is therefore both:

• a readable discipline publication
• a click-through navigation layer for the repository
• the root of the repository-book structure

HANDBOOK OPERATING RULE

The handbook must remain dynamic and navigable.

This means:

• the root handbook file links to handbook chapters
• handbook chapters synthesize stream domains
• stream indexes expose the authoritative artifact sets
• the root handbook links directly to primary stream artifacts
• stream artifacts remain authoritative in their own directories
• the handbook evolves by updating links and chapter structure rather than duplicating source content
