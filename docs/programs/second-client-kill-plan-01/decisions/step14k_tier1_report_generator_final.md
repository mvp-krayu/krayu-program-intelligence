## Changes to `scripts/pios/lens_report_generator.py`

### New global

- `_CLIENT_DISPLAY_NAMES` dict: maps client UUIDs to display names
- `_get_client_display_name(publish_safe)`: returns client name from dict or "Client Environment"

### New loaders

- `_load_psig_projection()` — reads `41.x/signal_projection.json` from run dir
- `_load_pressure_zone_projection()` — reads `41.x/pressure_zone_projection.json`
- `_load_binding_envelope()` — reads `binding/binding_envelope.json`

### New helpers

- `_compute_structural_metrics(binding)` — derives dep_load, edge_to_node, containment_density from binding summary
- `_compute_decision_model(metrics, psig_proj, gauge)` — computes structural_risk, evidence_completeness, decision

### New SVG builder

- `_build_tier1_topology_svg_generic(topology_domains, pz_proj)` — dynamic N-domain SVG from topology + zone projection

### Evidence brief (`_build_tier1_evidence_brief`)

- Signature extended: `psig_proj`, `pz_proj`, `metrics`, `decision_model` parameters
- `use_psig = psig_proj is not None` gates all second-client paths
- Client name: `_get_client_display_name(publish_safe)`
- SVG: `_build_tier1_topology_svg_generic` for `use_psig=True`
- Signal cards: PSIG-001/002/004/006 from `active_conditions_in_scope`; PSIG-003 NOT_ACTIVATED entry; structural metrics card
- Pressure zone block: rendered from `zone_projection`
- All BlueEdge paths preserved in `else` branches

### Narrative brief (`_build_tier1_narrative_brief`)

- Signature extended: same new parameters
- `use_psig` flag gates all second-client paths
- BlueEdge-only vars (`fleet_conn`, `vehicle_alert`, `driver_session`) gated on `not use_psig`
- Pressure zones section (§03): rendered from `zone_projection` when `use_psig=True`
- Signal table (§02): PSIG signal rows + dep_load/edge_to_node rows from `metrics`
- Known vs Unknown Boundary (§04): second-client unknown dimensions derived from `decision_model.not_activated`, `exec_evaluated`, `blind_spot_active`
- Decision Posture (§06): `decision_basis` block added (structural_risk / evidence_completeness / decision); narrative clarifies INVESTIGATE = evidence incompleteness, not structural instability
- All BlueEdge paths preserved in `else` branches

### `generate_tier1_reports()`

- Calls `_load_psig_projection()`, `_load_pressure_zone_projection()`, `_load_binding_envelope()`
- Calls `_compute_structural_metrics(binding)` and `_compute_decision_model(metrics, psig_proj, gauge)`
- Passes `psig_proj, pz_proj, metrics, decision_model` to both builder functions
