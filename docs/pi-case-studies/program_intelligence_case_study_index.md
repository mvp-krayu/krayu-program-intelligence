Program Intelligence Case Study Index Krayu --- Program Intelligence
Case Studies

Purpose

This document provides the index of all Program Intelligence case
studies produced within Stream 60 of the Krayu Program Intelligence
project.

The index catalogues the programs analyzed, the execution signals
detected, and the resulting Program Intelligence insights.

The objective is to enable cross-case analysis and identify recurring
execution signal patterns across different program environments.

All case studies referenced in this index are stored under:

docs/case-studies/

------------------------------------------------------------------------

Index Structure

Each case study entry documents:

• Program analyzed • Execution environment • Signals detected • Key
Program Intelligence insights • Artifact reference

Role of the Case Study Index

The index enables the Program Intelligence discipline to accumulate
empirical observations across programs.

By cataloguing signals across case studies, the discipline can identify:

• recurring execution patterns • common signal correlations • systemic
delivery behaviors

This accumulation of empirical evidence strengthens the analytical power
of the Program Intelligence framework.
