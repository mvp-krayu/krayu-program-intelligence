# Template — Invariant Amendment

## Instructions

Use this template when amending an existing invariant.

Rules:
- Locate the invariant in `/docs/brain/canonical/04_INVARIANTS.md`
- Append the amendment block directly below the existing invariant statement
- Do NOT delete or overwrite the original statement — it is preserved for lineage
- If the change is a scope expansion, state exactly what new layers or contexts now fall under this invariant
