## Governing Doctrine

- Evidence First — no output without traceable, anchored evidence
- Determinism — same input produces same output, no hidden logic
- Fail-Closed — ambiguity stops execution, does not resolve by assumption
- Layer Purity — no layer may contaminate another, ownership is fixed
- CREATE_ONLY lineage — evidence once written is not mutated
