# Program Intelligence Handbook

Krayu --- Program Intelligence Discipline

This handbook provides the structured navigation layer for the Program
Intelligence discipline developed within the Krayu Program Intelligence
project.

The handbook does **not duplicate content** produced by the individual
project streams.\
Instead, it references the authoritative artifacts produced in each
stream directory.

The Program Intelligence Handbook serves as the **navigation layer of
the discipline**.

discipline developed within the Krayu Program Intelligence project.
