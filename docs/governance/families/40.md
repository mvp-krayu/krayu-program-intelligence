## PURPOSE

PiOS core deterministic runtime — from governed telemetry through signal computation, condition activation, diagnosis, intelligence synthesis, delivery, feedback, and orchestration. This is the computational engine layer.

---

## STANDARD INVARIANTS

- Evidence-first: no output without traceable input
- Deterministic transformation only — same input produces same output
- Bounded stage-to-stage inputs — no cross-stage leakage
- No downstream artifact used as upstream evidence
- No speculative values without explicit traceability declaration
