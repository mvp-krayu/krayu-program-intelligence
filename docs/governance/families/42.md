## PURPOSE

Consumption and interpretation layer for query execution, executive narrative rendering, delivery packaging, topology exposure, and demo-facing governed output surfaces. 42.x streams translate engine outputs into inspectable, navigable, human-readable forms.

---

## STANDARD INVARIANTS

- Render only governed upstream truth — no synthetic metrics in UI
- Drill-down must preserve traceability to source signals
- Executive readability must not destroy evidence lineage
- Adapter outputs must declare value_source when multiple sources are possible
