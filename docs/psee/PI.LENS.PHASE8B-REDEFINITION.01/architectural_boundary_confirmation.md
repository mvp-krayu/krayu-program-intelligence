## Boundary: Pipeline Layer vs. Representation Layer

### Pipeline Layer (Phases 1–8b + 9)

Produces data and evidence artifacts. All outputs are:
- Deterministic
- Evidence-grounded
- Run-local (clients/<client>/psee/runs/<run_id>/)
- Independent of app infrastructure

Pipeline completion criterion: `vault_readiness.json` status = `READY`

### Representation Layer (separate invocation)

Consumes pipeline outputs. Produces HTML surfaces.
- Invoked explicitly via CLI (`lens_report_generator.py`) or LENS UI
- May depend on app vault (`vault_index.json`) for graph layout (Tier-2)
- Not a pipeline gate
- Not required for pipeline completion
