## 0. PURPOSE

This document is the consolidated cross-reference between LENS and the nine reference families.

It exists so that during a refinement iteration the reviewer can answer, in one document, the question: **"Which reference families am I aligned with, which am I drifting from, and where am I overshooting?"**

The deep-dives are authoritative for *what* each reference contributes. This matrix is authoritative for *comparing across the set*.

## 4. WHERE LENS LIVES

LENS is the *intersection* of the contribution columns and the *complement* of the avoid columns.

In one sentence:

> LENS is *Palantir's gravity* + *Bloomberg's hierarchy discipline* + *visionOS's restraint and depth* + *Linear's calm* + *Raycast's chrome economy* + *Arc's atmospheric continuity* + *Stripe / Vercel's precision* + *NASA / SpaceX's calm under consequence* + *FT's editorial pacing* — minus all of their respective avoid clauses.

This intersection is narrow. It is also the entire design space. There is nothing beyond the intersection that LENS should be reaching for.

## 5. ITERATION USE

During a refinement iteration:

1. Pick the rubric axis that failed (from `VISUAL_EVALUATION_RUBRIC.md`).
2. Read the contribution matrix row corresponding to that axis.
3. Identify the primary contributors (★).
4. Open those deep-dives.
5. Check the surface against the *borrow* sections.
6. Check the surface against the *avoid* sections.
7. Refine.

This procedure ensures iterations are reference-anchored, not taste-anchored.

---

## 6. DRIFT DETECTION

When LENS feels "off" but no specific axis is failing the rubric, run a drift check:

1. Show the surface to a fresh reviewer.
2. Ask: "What does this remind you of?"
3. If the answer names *one* reference family heavily, LENS has drifted toward that family.
4. Re-balance against the contribution matrix.

LENS that "looks like Palantir" is over-borrowing from family 1. LENS that "looks like Linear" is over-borrowing from family 4. The intersection is restored by re-engaging the under-used families.
