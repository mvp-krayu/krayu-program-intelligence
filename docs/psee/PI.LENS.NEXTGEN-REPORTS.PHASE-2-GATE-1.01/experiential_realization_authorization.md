## FORBIDDEN

The following are **permanently forbidden** in experiential realization and workspace evolution unless a new formally governed contract explicitly authorizes and constrains them:

### Free-Form AI
- AI-generated text in any executive surface
- Live language model calls within rendering pipeline
- Probabilistic confidence statements
- AI-generated propagation explanations

### Chatbot UX
- Conversational interface within any reporting surface
- Prompt input fields visible to executive users
- AI assistant avatars or personalization
- Suggested action generation

### Prompt-First UX
- Any UX flow that begins with a text prompt rather than evidence
- NLQ (natural language query) surfaces in Phase-2 rendering

### Uncontrolled Topology Exploration
- Free navigation within topology beyond committed propagation paths
- Topology editing or flagging interactions
- Dynamic domain discovery within reporting surfaces

### Live Inference
- Runtime propagation computation
- Dynamic readiness reclassification
- Evidence supplementation at render time
- Live qualifier reinterpretation

### Semantic Mutation
- Any rendering layer operation that modifies committed evidence
- Label overrides outside the governed vocabulary mappings
- Custom qualifier class definitions not in Q-taxonomy

### AI-Generated Executive Conclusions
- Summary text generated by AI in any reporting surface
- Executive decisions inferred by the rendering layer
- Causal explanations not present in committed narrative_block

### Animated Propagation Flow (VIS-PROP-02)
- Animated edges or arrows on propagation paths
- Directional flow animation (pulse, movement, trajectory)
- Edge weight visualization
- Consumer-style dependency graph animation

### Qualifier Chip Suppression (VIS-QUAL-02)
- Hiding qualifier chips (Q-01..Q-03) for aesthetic reasons
- Reducing qualifier chip visual prominence below readiness badge
- Removing Q-04 absence notice "Signal intelligence withheld from this view."
